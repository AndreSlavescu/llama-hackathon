# llama-hackathon

## BACKEND

create a `.env` file with the contents of [`.env.example`](.env.example)

then run:

```
source .env
```

```
pip install -r requirements.txt
fastapi dev main.py
```

### Build llm_generation dependencies
```
chmod +x llm_generation/build.sh
```

```
./llm_generation/build.sh
```
